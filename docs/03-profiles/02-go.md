```
Primary:     ocg/kimi-k2.6 → ocg/qwen3.6-plus → ocg/glm-5.1
Small:       oc/deepseek-v4-flash-free → kr/glm-5 → oc/north-mini-code-free
```

**Catatan:** Small model tetap pakai gratis.

## Kelebihan

- ✅ Kualitas model lebih tinggi
- ✅ Support vision (beberapa model)
- ✅ Response lebih cepat
- ✅ Auto-fallback ke gratis jika quota habis

## Kekurangan

- ⚠️ Limited quota
- ⚠️ Perlu API key
- ⚠️ Quota reset per bulan

## Setup

### Via Script

```powershell
.\scripts\start.ps1 -Profile go
```

### Via OpenCode

```
/start-go
```

### Manual

```powershell
Copy-Item "profiles\go\opencode.jsonc" "$env:USERPROFILE\.config\opencode\opencode.jsonc"
```

## Config

Config tersedia di `profiles/go/opencode.jsonc`.
