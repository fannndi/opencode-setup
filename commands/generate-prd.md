# Generate PRD

Ubah ide Anda menjadi Product Requirements Document (PRD) otomatis via AI.

## Instructions

```powershell
C:\Users\FANNNDI\Documents\opencode-setup\scripts\generate-prd.ps1 -Idea "deskripsi aplikasi Anda"
```

## Contoh

```powershell
.\generate-prd.ps1 -Idea "Aplikasi kasir untuk toko kelontong dengan fitur stok dan laporan harian"
```

## Output

- File `prd.md` di project root
- Tech stack recommendation
- Feature list
- Timeline estimation
- Skills needed
