# Token Stats

Tampilkan estimasi token usage, session stats, dan rekomendasi.

## Instructions

```powershell
.\scripts\token-tracker.ps1
```

## Output

- Model availability
- Combo status
- Token usage estimate
- Cost estimate
- Session stats (profile, project, stack, iteration)
- Auto-compact warning if >100K tokens
