## Usage in analyze-project.ps1

The `analyze-project` script uses this catalog to:

1. **Detect stack** → match indicator files
2. **Load skills** → use "By Stack" section
3. **Apply rules** → use "Rules" from stack entry
4. **Generate config** → build instructions array

### Example: dart-flutter

```powershell
# From skill-list.md
$stack = "dart-flutter"
$coreSkills = @("tdd-workflow", "security-review", "coding-standards", "verification-loop")
$projectSkills = @("dart-flutter-patterns")
$rules = @("common", "dart")
$allSkills = $coreSkills + $projectSkills

# Generate config
$config.instructions = $allSkills | ForEach-Object {
    "C:/Users/FANNNDI/Documents/opencode-setup/ecc/skills/$_/SKILL.md"
}
```
